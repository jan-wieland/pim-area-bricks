# jan-wieland/pim-area-bricks

Pimcore Area Bricks Bundle mit Webpack Encore, SCSS, Tailwind CSS 4 und JS.

## Installation

```bash
composer require jan-wieland/pim-area-bricks
```

## Assets bauen

```bash
npm install
npm run build
```

Oder während der Entwicklung:

```bash
npm run watch
```

## Anforderungen

- PHP >= 8.1
- Pimcore ^11.0
- Node.js >= 18
